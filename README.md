# 🍽️ ChainChaser

A web app for tracking visits to every location of restaurant chains. Pick a chain, see all locations on a map, mark the ones you've visited, and track your completion progress.

👉 **Try it live:** [lucasabella.github.io](https://lucasabella.github.io)

## About

This is an open-source project — feel free to explore the code, learn from it, or play around with it! Currently focused on Dutch restaurant chains.

## Tech Stack

- **Frontend:** React + Vite
- **Map:** Leaflet (via react-leaflet)
- **Storage:** localStorage (no account needed)
- **Hosting:** GitHub Pages
